    Module phpBB pour utiliser les utilisateurs du forum dans MushRaider

    Projet maintenu par Kyah

    Présenstation

    Ce projet s'installe comme un MODS pour phpBB v3.0.X (une MaJ pour la version 3.1.X est en préparation) et permet aux membres d'un forum de se connecter avec leur Nom d'utilisateur et Mot de passe sur une instance MushRaider associée

    Pré-requis

    Avant d'installer le module, il vous faut :

    • Un forum phpBB en version 3.0.X installé et fonctionnel ( cf. Forum phpbb-fr )
    • Un raidplanner MushRaider installé en version 1.4+ ( cf. MushRaider.com )
    • Un accès à l'hébergement du forum pour être en mesure de déposer des fichiers (vous avez probablement déjà ça si vous avez un forum installé )
    • Le zip du module disponible ici : https://github.com/Kyah/mushraider_phpbb_bridge/zipball/master

    Installation du module phpBB

    Extraction de l'archive

    • Extrayez l'archive téléchargée ci-dessus
    • Accedez au contenu du dossier extrait précedment

    Dépot des fichiers

    • Connectez vous au FTP de l'hébergement et positionnez vous dans le dossier racine du forum
    • Copiez l'intégralité des fichiers et dossiers contenu dans le dossier root qui a été extrait précédemment

    Exécution de l'installation

    • Rendez vous à la page d'installation du module sur le forum (par exemple http://www.maguilde.fr/forum/install_mushraider.php )
    • Laissez les valeurs par défaut et appuyez sur Envoyer
    • La page suivante demande un validation, appuyez sur Oui
    • Et voilà, le module est installé!

    Suppression du fichier devenu inutile

    • Reconnectez vous au FTP de l'hébergement et positionnez vous dans le dossier racine du forum
    • Supprimez le fichier install_mushraider.php (cela éviteras que des personnes mal intentionnées intéragissent avec le module)

    Configuration de phpBB

    • Connectez vous au panneau d'administration du forum
    • Trouvez l'onglet .Mods en haut de la page et cliquez dessus
    • A gauche, en dessous de la légende MUSHRAIDER BRIDGE se trouvent les deux pages de configuration du module MushRaider

    Général

    Cette page permet de gérer la configuration de base du module. Voici les options disponibles :
    • Activer le Bridge MushRaider? : Permet de désactiver le module si il venait a ne plus être utile.
      • ATTENTION cela ne change pas la configuration de MushRaider

    Permissions

    Cette partie permet d'associer les groupes existant sur le forum aux 3 rôles possibles dans le raidplanner :
    • Admin
    • Officier
    • Membre
    Des valeurs sont remplies par défaut mais il est conseillé de personnaliser ces paramètres en fonction de l'organisation mise en place sur le forum. Il est possible d'associer plusieurs groupes à un même rôle, pour cela il suffit de maintenir la touche CTRL du clavier tout en cliquant sur les différents groupes.

    ATTENTION Si un groupe se retrouve dans plusieurs rôles, seul celui offrant le plus de liberté sera choisi. A savoir : Admin > Officier > Membre

    Une fois ces valeurs modifiées, appuyez sur Envoyer

    Configuration de MushRaider

    • Connectez vous avec un compte Administrateur au raidplanner
    • Accédez au panneau d'Admin
    • Dans les onglets de gauche sélectionnez Paramètres puis Intégration
    • Cochez la case Intégrer MushRaider dans votre site/forum actuel
    • Dans la case Adresse du site tier indiquez l'url suivie de /mushraider_bridge/login.php. Exemple : http://www.maguilde.fr/forum/mushraider/login.php
    • Dans la case Clef secrète indiquez une clé que vous avez choisi précedment dans la configuration du forum.
    • Appuyez sur Sauvegarder
    Voilà c'est terminé! Désormais seul les comptes du forum peuvent se connecter au raidplanner!

    ATTENTION Si vous votre version de MushRaider est inférieure a la 1.3.5 et que vous aviez des comptes déjà existant dans le raidplanner et que vos utilisateurs ont le même nom sur le forum mais que leurs mot de passe diffèrent entre les 2 outils. Il faudra malheureusement supprimer leur compte et qu'ils recréé tout leurs personnages
